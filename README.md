# Neon Forum (PHP + MySQL)

A full-stack discussion forum built using PHP, MySQL, Bootstrap, and JavaScript.

## Features
- User registration & login system
- Profile management
- Create, edit, delete topics
- Comment system with live updates
- Admin controls:
  - Delete topics & comments
  - Suspend users
- Activity log system

## Technologies Used
- PHP
- MySQL
- Bootstrap
- JavaScript (AJAX)

## How to Run
1. Install XAMPP
2. Place project in htdocs
3. Start Apache & MySQL
4. Import database (forum_db)
5. Visit http://localhost/forum_project
